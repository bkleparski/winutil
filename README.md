# WinUtil

Zestaw narzedzi do administracji i optymalizacji **Windows** — naprawy, tweaki, zarzadzanie systemem z poziomu PowerShell.

## Uruchomienie

```powershell
irm go.ebartnet.pl/winutil | iex
```

> **Wymagane:** terminal PowerShell uruchomiony jako **Administrator**.

---

## Zawartość

Plik `winutil.ps1` zawiera zestaw funkcji do administracji Windows:

- Naprawa i optymalizacja systemu
- Zarzadzanie ustawieniami Windows
- Narzedzia diagnostyczne
- Administracja z poziomu CLI

---

## Bezposrednie pobranie

```powershell
iwr -useb https://raw.githubusercontent.com/bkleparski/winutil/main/winutil.ps1 | iex
```

---

## Wymagania

- Windows 10 / 11
- PowerShell 5.1+
- Uprawnienia Administratora
